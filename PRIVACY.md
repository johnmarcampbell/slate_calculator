# Privacy Policy — Slate Calculator

_Last updated: July 11, 2026_

Slate Calculator is a Chrome extension that evaluates typed math expressions, plots functions, and keeps a history of your calculations.

## Data collection

Slate Calculator does **not** collect, transmit, sell, or share any data. There are no analytics, no tracking, no remote servers, and no network requests of any kind.

## Local storage

The extension stores the following on your device only, using Chrome's `chrome.storage.local` API:

- Calculation history (up to 80 entries)
- Preferences: theme, angle mode (radians/degrees), number format settings, graph ranges, active view, and the current expression draft

This data never leaves your browser. You can erase the history at any time with the clear-history button in the popup, and all stored data is removed when you uninstall the extension.

## Permissions

The extension requests a single permission, `storage`, which is used solely for the local persistence described above.
